Notes:

  1. Moderation Commands Require a Channel set with x;modchannel to send Ban/Softban/Kick/Unban/Warn Logs
  2. To use Member Join/Leave Logging, set a channel with x;memberchannel, a custom message can be set with x;membermsg
  3. You can set a special role that is able to use Moderation commands (excluding lockdown) with x;staffrole
  4. Use x;help to View a Command List
  5. Visit my Home Server for more support, or for updates, or if you just want to hang out.
